CSS3 Transformations

The CSS Transform property provides a way to transform div elements by
moving them based on different values. The div element can be scaled,
moved to different points on an axis, rotated, skewed, and others.



     body {
            padding: 5em;
        }
        div {
            width: 300px;
            height: 100px;
            background-color: darkkhaki;
            border: 1px solid black;
        }
        #demo2:hover {
            background-color: lightslategray;
            transform: scale(.5,.5);
        }
        #demo1:hover {
            background-color: aquamarine;
            transform: rotate(25deg);
        }
        #demo3:hover {
            background-color: beige;
            transform: translate(400px,-175px);
        }

Hover over the DIVs and watch them change!

You will notice that the change is not fluid.
