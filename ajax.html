AJAX Interactions


AJAX = Asynchronous Javascript and XML Server Data Exchange Allows part of a webpage to be updated, without reloading the entire page (asynchronous) --This is faster, and therefore more preferable to reloading Examples: Google Maps (can drag without reloading), Mail services (new email appears without reload), Facebook Data is stored on the server in XML format --note: AJAX is not supported by older browsers, such as IE4 or earlier.

XMLHttpRequest Object: //for IE7, Firefox, Chrome, Safari, etc. ajaxRequest = new XMLHttpRequest();

    function loadXMLDoc(){
        var xmlhttp;
        if (window.XMLHttpRequest)
            {// code for IE7+, Firefox, Chrome, Opera, Safari
            xmlhttp=new XMLHttpRequest();
        }
        else {// code for IE6, IE5
            xmlhttp=new ActiveXObject("Microsoft.XMLHTTP");
        }
        xmlhttp.onreadystatechange=function()
        {
        if (xmlhttp.readyState==4 && xmlhttp.status==200)
        {
        document.getElementById("myDiv").innerHTML=xmlhttp.responseText;
        }
    }
    xmlhttp.open("GET","sample.txt",true);
    xmlhttp.send();
    }
