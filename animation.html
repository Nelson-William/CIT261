CSS3 Animation

Animations

Animations change the style of an element over a specified period of time. This requires keyframes.

@Keyframes

Keyframes dictate what style an element will have at different points during the animation.
A keyframe can use a from and to, or indicate a series of percentages.
Keyframes are used the type of animation, and you can even implement many of the CSS3 Transform properties.
Other animation properties, such as timing, are used on the animation's container.

Container Properties

animation-name: calls the animation indicated by the keyframe.
animation-duration: how many seconds or milliseconds in one cycle.
animation-timing-function: animation speed curve.
animation-delay
animation-iteration-count: number of times animation should be repeated. Can be infinite.
animation-direction: alternate or normal.

animation-fill-mode: indicates a style when animation is finished, or during delay.
animation-play-state: dictates whether animation is paused.
animation: shorthand for setting animation properties except play-state and fill-mode.

Timing Functions

linear
ease
ease-in
ease-out
ease-in-out
cubic-bezier(n,n,n,n) 


    @keyframes example1{
        from {background-color: blue;}
        to {background-color: orange;}
    }
    #example1 {
        animation-name: example1;
        animation-duration: 10s;
        animation-timing-function: ease;
        animation-delay: 3s;
        animation-iteration-count: infinite;
        animation-direction: alternate;
    }
    div {
        animation: example1 10s ease 3s infinite alternate;
    }


    @keyframes example2{
        0%  {background-color: red; left:0px; top:0px;}
        25% {background-color: green; left:50px; top:50px;}
        50% {background-color: yellow; left:0px; top:50px;}
        75% {background-color: blue; left:50px; top:0px;}
        100%{background-color: orange; left:0px; top:0px;}
    }
    #example2 {
        animation: example2 5s linear 2s infinite alternate;
    }


    @keyframes example3{
        0%  {background-color: gray; left:0px; top:0px;}
        25% {
            transform: rotate(90deg);
            left: 50px;
        }
        50% {
            transform: rotate(180deg);
            left: 100px;
        }
        75% {
            transform: rotate(270deg);
            left: 150px;
        }
        100% {
            transform: rotate(360deg);
            left: 200px;
        }
    }
    #example3 {
        animation: example3 7s ease-in-out 2s infinite alternate;
    }
