CSS3 Transitions

What are CSS3 Transitions?

Transitions enable the change of a property from one value to another over a defined period of time. This requires: a time period, and an effect.

Transition Properties

  • transition: summary property for below four properties.
  • transition-delay: seconds of delay before transition begins.
  • transition-duration: seconds of time transition takes to complete.
  • transition-property: name of CSS property the transition is for.
  • transition-timing-function: sets timing property of the transition.
    • ease
    • linear
    • ease-in
    • ease-out
    • ease-in-out
    • cubic-bezier()

Sample Code:


            #example1{
                transition: background-color 3s;
            }
            #example1:hover{
                background-color: #C0362C;
            }

Browser Support

-webkit- for Chrome and Safari
-moz- for Firefox.
Necessary prefixes for proper browser compatibility.

transition: width 2s;
-webkit-transition: width 2s;/*Safari,Chrome*/
-moz-transition: width 2s;/*Firefox*/


            #example2{
                transition: width 3s;
                -webkit-transition: width 3s;
                -moz-transition: width 3s;
            }
            #example2:hover{
                width: 350px;
            }

Changing Multiple Values

It is possible to change more than one value at at time.


            #example3{
                background-color: 3s;
                transition: width 3s, height 4s;
                transition-timing-function: ease-in;
                -webkit-transition-timing-function: ease-in;
                -webkit-transition: width 3s, height 4s;
                -moz-transition: width 3s, height 4s;                
            }
            #example3:hover{
                background-color: aqua;
                width: 300px;
                height: 175px;
            }

Adding Other CSS3 Changes

It is possible to add CSS Transform properties to the transition property.


            #example4{
                background-color: 3s;
                transition: width 3s, height 3s, transform 3s;
                transition-timing-function: cubic-bezier(0,1.7,1,-1.9);
                -webkit-transition-timing-function: cubic-bezier(0,1.7,1,-1.9);
                -moz-transition-timing-function: cubic-bezier(0,1.7,1,-1.9);
                -webkit-transition: width 3s, height 3s, -webkit-transform 3s;
                -moz-transition: width 3s, height 3s, -moz-transform 3s;
            }
            #example4:hover{
                width: 200px;
                height: 200px;
                background-color: coral;
                transform: rotate(300deg);
                -webkit-transform: rotate(300deg);
                -moz-transform: rotate(300deg);
            }
