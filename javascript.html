JavaScript Objects

A JavaScript Object can be understood in terms of a real life object. It can have properties that describe it, and methods or actions that it can perform. A method is treated similar to a function.

Like a variable, an object can have a value assigned to it. However, an object can have multiple values. An object is created by assigning a value to it, such as var guitar = {type:'electric', strings:'six', color:'red'};

An object can be called in two ways,
guitar.type, or guitar[type]

For a method, it is accessed by:
guitar.toneUp() 


var myGuitar = {type:'acoustic', strings:' six ', color:'brown', name:'Martin'};
